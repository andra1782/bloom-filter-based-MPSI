### Bloom Filter-based Multi-Party Private Set Intersection (MPSI)
Master's Thesis CS Computer Science, Delft University of Technology

## Pre-requisites
```
sudo apt update
sudo apt install cmake
sudo apt install build-essential cmake
sudo apt install libntl-dev libgmp-dev
```

## Instructions
```
cd bf-mpsi-baseline/
mkdir build
cd build
cmake ..
cmake --build . --config Release
./bf_mpsi_baseline
```
